# AI Mastery Plan 2026 🚀

A **12-week aggressive, project-driven** self-paced learning journey to master **AI, Machine Learning, LLMs, RAG, Agentic AI, and Production System Design**.

**Goal**: Build 12 substantial, portfolio-worthy AI systems while developing strong engineering fundamentals and production thinking.

## Tech Stack

- **LLMs**: Ollama (Meta Llama / Qwen2.5) + Gemini (fallback)
- **Frameworks**: LangChain, LangGraph, CrewAI
- **Vector DBs**: Chroma, FAISS (later PGVector)
- **Evals & Observability**: Ragas, LangSmith, DeepEval
- **Frontend**: Streamlit / Gradio
- **Backend**: FastAPI
- **Cloud**: AWS Bedrock (optional), Docker
- **Development**: Dev Containers, pre-commit hooks

## Learning Pillars

- **deeplearning.ai** courses (Andrew Ng)
- **Stanford Online** lectures (CS229, CS224N, CS230)
- **LangChain** + **Anthropic** best practices
- **John Crickett** Coding Challenges
- **Fernando Franco** engineering articles
- Heavy emphasis on **evaluations**, **system design**, and **real projects**

## Project Standards
- Clean architecture & modular code
- Comprehensive evaluations + LangSmith tracing
- Docker support
- Detailed README per project with architecture diagrams
- Performance analysis and trade-off documentation

## How to Run

1. **Clone the repo**
   ```bash
   git clone https://github.com/moijes12/ai-mastery.git
   cd ai-mastery
   ```
2. Use Dev Container (recommended) or:
   ```bash
   ollama serve
   ollama pull qwen2.5:7b
   ```
3. Environment
   ```bash
   cp .env.example .env
   # Fill in your keys
   ```
4. Navigate to weekly projects:
   ```bash
   cd projects/week1-ticket-system
   ```
